CChartViewer.isInMouseMoveEvent

Usage

bool isInMouseMoveEvent();

Description

Checks if the current code is executing in a MouseMoveChart or MouseMovePlotArea event.

Arguments

None

Return Value

True if the current code is executing in a CVN_MouseMoveChart message or CVN_MouseMovePlotArea message , otherwise false.
