ViewPortManager.getZoomInHeightLimit

Usage

double getZoomInHeightLimit();

Description

Gets the viewport height at maximum zoom in for mouse zoom in actions.

Arguments

None

Return Value

The viewport height at maximum zoom in. See ViewPortManager.setZoomInHeightLimit for the meaning of this value.
