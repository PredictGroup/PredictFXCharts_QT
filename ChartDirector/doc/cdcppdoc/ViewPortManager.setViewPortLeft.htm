ViewPortManager.setViewPortLeft

Usage

void setViewPortLeft(double left);

Description

Sets the position of the left side of the viewport.

The position of the viewport left side is its distance from the left side of the underlying surface, as a fraction of the width of the underlying surface. It should be between 0 and 1.

Arguments

ArgumentDefaultDescription
left(Mandatory)The position of the left side of the viewport.

Return Value

None
