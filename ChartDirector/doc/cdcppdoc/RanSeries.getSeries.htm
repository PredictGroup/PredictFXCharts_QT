RanSeries.getSeries

Usage

DoubleArray getSeries(int len, double minValue, double maxValue);

Description

Gets an array of independent random numbers.

Arguments

ArgumentDefaultDescription
len(Mandatory)The number of random values to get.
minValue(Mandatory)The minimum value of the random number.
maxValue(Mandatory)The maximum value of the random number.

Return Value

An array of independent random numbers.
