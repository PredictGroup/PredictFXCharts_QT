ChartDirector 6.0 (C++ Edition)

CChartViewer.CVN_MouseLeavePlotArea

Description

This is a windows notification code that is sent to the parent window through the WM_COMMAND message when the mouse leaves the extended plot area.

See CChartViewer.setPlotAreaMouseMargin on what is the extended plot area.
