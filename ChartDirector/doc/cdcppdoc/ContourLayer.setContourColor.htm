ContourLayer.setContourColor

Usage

void setContourColor(int contourColor, int minorContourColor = -1);

Description

Sets contour line colors.

Arguments

ArgumentDefaultDescription
contourColor(Mandatory)The major contour line color. If this method is never called, the default contour line color is semi-transparent black (80000000).
minorContourColor-1The minor contour line color. -1 means the minor contour line color is the same as the major contour line color.

Return Value

None
