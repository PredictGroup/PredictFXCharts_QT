ViewPortControlBase.setDragInsideToMove

Usage

void setDragInsideToMove(bool b);

Description

Enables or disables dragging inside the viewport to move it.

Arguments

ArgumentDefaultDescription
b(Mandatory)True to enable the behaviour, false to disable it.

Return Value

None
