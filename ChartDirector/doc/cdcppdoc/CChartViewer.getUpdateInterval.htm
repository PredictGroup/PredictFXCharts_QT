CChartViewer.getUpdateInterval

Usage

int getUpdateInterval();

Description

Gets the minimum gap interval between two CVN_ViewPortChanged messages in milliseconds.

Arguments

None

Return Value

The minimum gap interval between two CVN_ViewPortChanged messages in milliseconds. See CChartViewer.setUpdateInterval for the meaning of this value.
