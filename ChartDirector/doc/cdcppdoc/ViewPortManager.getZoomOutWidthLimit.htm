ViewPortManager.getZoomOutWidthLimit

Usage

double getZoomOutWidthLimit();

Description

Gets the viewport width at maximum zoom out for mouse zoom out actions.

Arguments

None

Return Value

The viewport width at maximum zoom out. See ViewPortManager.setZoomOutWidthLimit for the meaning of this value.
