ViewPortControlBase.setDragBorderToResize

Usage

void setDragBorderToResize(bool b);

Description

Enables or disables dragging the viewport border to resize it.

Arguments

ArgumentDefaultDescription
b(Mandatory)True to enable the behaviour, false to disable it.

Return Value

None
