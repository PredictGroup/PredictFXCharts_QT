ChartDirector 6.0 (C++ Edition)

Chart.DirectionHorizontal

Declaration

static const int DirectionHorizontal;

Description

A constant equals to 0 to represent that the zoom and/or scroll orientation is horizontal.

This constant is used in CChartViewer.setZoomDirection, CChartViewer.setScrollDirection, QChartViewer.setZoomDirection and QChartViewer.setScrollDirection .
