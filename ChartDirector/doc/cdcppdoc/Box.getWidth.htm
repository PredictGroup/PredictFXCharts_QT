ChartDirector 6.0 (C++ Edition)

Box.getWidth

Usage

int getWidth() const;

Description

Gets the width of the box.

In some cases, the width of a box may be dynamically determined. An example is the width of an LegendBox. For these cases, the width is undefined until the legend box or the entire chart has been laid out (using BaseChart.layout or BaseChart.layoutLegend), or the chart image has been drawn (eg. using BaseChart.makeChart, BaseChart.makeChart2 or BaseChart.makeChart3).

Arguments

None

Return Value

The width of the box in pixels.
