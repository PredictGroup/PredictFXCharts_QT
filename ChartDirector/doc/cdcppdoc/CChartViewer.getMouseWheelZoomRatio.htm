CChartViewer.getMouseWheelZoomRatio

Usage

double getMouseWheelZoomRatio();

Description

Gets the magnification ratio for each mouse wheel event.

Please refer to CChartViewer.setMouseWheelZoomRatio for the usage of this ratio.

Arguments

None

Return Value

The magnification ratio.
