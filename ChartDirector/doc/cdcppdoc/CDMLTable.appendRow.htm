CDMLTable.appendRow

Usage

TextBox *appendRow();

Description

Appends a row to the bottom of the table.

Arguments

None

Return Value

A TextBox object representing the prototype of the cells in the new row. This may be used to fine-tune the appearance of the cells in the new row.
