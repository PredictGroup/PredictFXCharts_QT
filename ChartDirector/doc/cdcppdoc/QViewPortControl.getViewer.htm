QViewPortControl.getViewer

Usage

QChartViewer *getViewer();

Description

Gets the QChartViewer object provided by the previous QViewPortControl.setViewer method call.

Arguments

None

Return Value

The QChartViewer pointer provided by the previous QViewPortControl.setViewer method call.
