QChartViewer.setDefaultToolTip

Usage

void setDefaultToolTip(QString text);

Description

Sets the default tool tip to use when the mouse is over the QChartViewer but not over any hot spot.

Arguments

ArgumentDefaultDescription
text(Mandatory)The default tool tip text to use.

Return Value

None
