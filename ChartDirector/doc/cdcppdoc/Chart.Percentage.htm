ChartDirector 6.0 (C++ Edition)

Chart.Percentage

Declaration

static const int Percentage;

Description

A constant equals to 4 to represent the "Percentage" data representation method for multiple data sets.

This constant is used in XYChart.addBarLayer2, XYChart.addAreaLayer2 and Layer.setDataCombineMethod.
