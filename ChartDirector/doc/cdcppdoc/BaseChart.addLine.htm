ChartDirector 6.0 (C++ Edition)

BaseChart.addLine

Usage

Line *addLine(int x1, int y1, int x2, int y2, int color = LineColor, int lineWidth = 1);

Description

Adds a line to the chart.

Arguments

ArgumentDefaultDescription
x1(Mandatory)The x pixel coordinate of the first endpoint of the line.
y1(Mandatory)The y pixel coordinate of the first endpoint of the line.
x2(Mandatory)The x pixel coordinate of the second endpoint of the line.
y2(Mandatory)The y pixel coordinate of the second endpoint of the line.
colorLineColorThe color of the line.
lineWidth1The width of the line.

Return Value

A Line object representing the line added. You may use this object to fine- tune the appearance of the line.
